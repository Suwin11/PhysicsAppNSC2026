================= ธีมตัวอย่าง (dummy template) =================

วิธีใช้: ก็อปโฟลเดอร์นี้ทั้งอัน -> เปลี่ยนชื่อเป็นชื่อธีมของคุณ (เช่น night, ocean)
-> ชื่อโฟลเดอร์จะโผล่ใน Settings > Theme อัตโนมัติ

มีอะไรในนี้ (ตัวอย่างครบทุกโฟลเดอร์ แม้ยังไม่มีรูป):
  palette.json        <- สี canvas (แก้ค่าได้เลย) — มีตัวอย่างให้แล้ว
  background/          <- วาง background.png (ไม่มี = ผู้เล่นเลือกสีเอง)
  button/             <- เฟรมปุ่ม
    lock/             <- ปุ่มล็อก (ถ้าจะวาดแยกทุกปุ่ม)
  icon/               <- ไอคอน toolbar/control

ไม่ต้องใส่ครบก็ได้ — ไฟล์ไหนไม่มี ระบบดึงจาก _shared/ ให้
ธีมสีล้วน (ไม่มีรูปเลย) ก็ใช้ได้ แค่มี palette.json อย่างเดียว

รูปทั้งหมด ❌ ห้ามมีตัวอักษร
